Criar BACK-END

mkdir backend
cd backend
npm init -y
npm install express
npm install nodemon -D
npm install knex
npm install sqlite3
npx knex init
npx knex migrate:make create_ongs
npx knex migrate:latest		Executa e cria a tabela
npx knex				Lista comandos
npx knex migrate:rollback		Desfaz o ultimo comando
npx knex migrate:status		Mostra todas tabelas
npm install cors			Quem pode acessar



GET: Buscar/listar uma informação do back-end
POST: Criar uma informação no bk
PUT: Alterar uma informação no bk
DELETE: Deletar uma informação


Tipos de parâmetros:

Query Params: Parâmetros nnomeados enviados na roda após "?" (filtros, paginação)
Route Params: Parâmetros utilizados para identificar recursos
Request Body: Corpo da requisição, utilizado para criar ou alterar recursos


SQL: MySQL, SQLite, PostgreSQL, Oracle, Microsoft SQL Server
NoSQL: MongoDB, Couch, etc



Entidade:
		ONG
		Caso (insident)

Funcionalidades:
		Login de ONG
		Logout de ONG
		Cadastro de ONG
		Cadastrar novos casos
		Deletar casos
		Listar casos específicos de uma ONG
		Listar todos os casos
		Entrar em contato com a ONG (Whats, Email)
